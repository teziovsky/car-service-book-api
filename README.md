## Get started 🏁

Clone the project and go to the project directory

```bash
git clone git@github.com:teziovsky/car-service-book-api.git && cd car-service-book-api
```

Install dependencies

```bash
npm install
```

Start the application!

```bash
npm run start:dev
```
